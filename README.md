# AJS Classes

Иерархия классов персонажей для игры.

## Классы

- `Character` - базовый класс
- `Bowman` - лучник (атака: 25, защита: 25)
- `Swordsman` - мечник (атака: 40, защита: 10)
- `Magician` - маг (атака: 10, защита: 40)
- `Daemon` - демон (атака: 10, защита: 40)
- `Undead` - нежить (атака: 25, защита: 25)
- `Zombie` - зомби (атака: 40, защита: 10)
